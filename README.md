Supports Chess♯ family of variants in dart. Learn more about Chess♯ here:

https://chess-sharp.games/
https://chess-sharp.games/ChessSharp_Rules.pdf

## Usage

```dart
import 'package:chess_sharp_dart/dartchess.dart';
```
